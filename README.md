# CINEMA Premium: Система Бронювання Квитків 
CINEMA Premium — це веб-додаток, розроблений як курсова робота, що імітує онлайн-систему бронювання квитків у кінотеатр. Додаток надає користувачам можливість переглядати афішу, вибирати сеанси та місця на інтерактивній схемі залу, а також керувати своїм профілем та бронюваннями.

---

## Ключові Функції: 
Перегляд поточного розкладу сеансів на головній сторінці. Окрема сторінка з описом, тривалістю, жанром та списком доступних сеансів для конкретного фільму.
**Інтерактивне Бронювання:**
- Вибір місць на інтерактивній схемі залу.
- Автоматичний розрахунок загальної вартості.
- Обмеження на максимальну кількість місць (10 квитків за одне бронювання)

## Користувацький Кабінет:
Реєстрація та Авторизація (з хешуванням паролів за допомогою werkzeug.security).
Сторінка профілю з можливістю оновлення особистих даних.
Перегляд історії всіх бронювань.
Система сповіщень (системні, промо, про бронювання).
Управління Бронюванням: Перегляд деталей бронювання за унікальним кодом та можливість скасування активного бронювання.

## Тестові Дані Користувачів

## Використані технології:
- **Backend:** Python 3.О - сновна мова програмування.
- **Web Фреймворк:** Flask 2.3.3 - Мікро-фреймворк для розробки веб-додатка та API.
- **База Даних:** SQLite 3 - Легка, вбудована база даних для зберігання всіх даних (фільми, сеанси, місця, бронювання, користувачі).
- **Frontend:** Jinja2 - Шаблонізатор для динамічної генерації HTML-сторінок.
- **Стилі:** Bootstrap 5.1.3 + CSS3 - Адаптивний дизайн та унікальна сучасна тема з використанням золотих/срібних акцентів.
- **Інтерактив:** JavaScript (ES6) - Обробка логіки вибору місць, валідації та AJAX-запитів на сторінці бронювання.

### Кроки встановлення
```bash
# 1. Клонувати репозиторій
git clone git@github.com:AnastasiiaTsi/Cinema.git

# 2. Перейти в папку проекту
cd Cinema

# 3. Встановити залежності
pip install -r requirements.txt

# 4. Налаштувати базу даних
mysql -u root -p < database.sql

# 5. Запустити проект
python app.py
```
